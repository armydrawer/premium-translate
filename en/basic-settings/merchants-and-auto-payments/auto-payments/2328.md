## Module Settings

In the admin panel, create a new merchant in the "**Merchants**" ➔ "**Add Auto Payout**" section.

Select 2328 from the drop-down list in the "**Module**" field, enter a name for the module, and click "**Save**."

<figure><img src="../../../.gitbook/assets/image (306).png" alt=""><figcaption></figcaption></figure>

Fill in the specified authorization fields.

<figure><img src="../../../.gitbook/assets/image (307).png" alt=""><figcaption></figcaption></figure>

**Domain** — leave this field empty.

**Project ID** — the ID you copied earlier from your 2328 account.

**Payment API Key** — the payment key you copied earlier from your 2328 account.

**Payout API Key** — the payout key you copied earlier from your 2328 account.

## Special Fields

<div><figure><img src="../../../.gitbook/assets/image (318).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../../.gitbook/assets/image (340).png" alt=""><figcaption></figcaption></figure></div>

**Currency Code** (for payouts)**:**

* **Custom Fields (Order)** — use the currency code from the request (select **\[Receive] Currency Code**)
* **Custom Fields (Currency)** — use the [additional currency field](https://premium.gitbook.io/main/osnovnye-nastroiki/valyuty-i-napravleniya-obmena/dopolnitelnye-polya#dopolnitelnye-polya-dlya-valyuty) "**Receive**"
* **Custom Fields (Direction)** — use the [additional exchange direction field](https://premium.gitbook.io/main/osnovnye-nastroiki/valyuty-i-napravleniya-obmena/dopolnitelnye-polya#dopolnitelnye-polya-dlya-napravleniya-obmena)
* **Currency Code** — manually select the payout currency.

<div><figure><img src="../../../.gitbook/assets/image (343).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../../.gitbook/assets/image (364).png" alt=""><figcaption></figcaption></figure></div>

**Network** (for cryptocurrencies)**:**

* **Custom Fields (Currency)** — use the [additional currency field](https://premium.gitbook.io/main/osnovnye-nastroiki/valyuty-i-napravleniya-obmena/dopolnitelnye-polya#dopolnitelnye-polya-dlya-valyuty) "**Receive**"
* **Custom Fields (Direction)** — use the [additional exchange direction field](https://premium.gitbook.io/main/osnovnye-nastroiki/valyuty-i-napravleniya-obmena/dopolnitelnye-polya#dopolnitelnye-polya-dlya-napravleniya-obmena)
* **Network** — manually select the network.

**Converting from another currency (specify code)** — use conversion from the selected currency (usually USDT) to the payout currency just before the payout is processed.
