Register for the FeeCutEx service via the [Telegram bot](https://t.me/feecutex_bot) and provide the service representative with your username to receive your API keys.

## Module Settings

In the admin panel, create a new merchant in the "**Merchants**" ➔ "**Add Merchant**" section.

Select FeeCutEx from the dropdown menu in the "**Module**" field, enter a name for the module, and click "**Save**."

<figure><img src="../../../.gitbook/assets/изображение (229).png" alt=""><figcaption></figcaption></figure>

Fill in the required authorization fields.

<figure><img src="../../../.gitbook/assets/изображение (230).png" alt=""><figcaption></figcaption></figure>

**Domain** — leave this field empty.

**Secret Key** — the API key provided to you earlier by the FeeCutEx representative.

**User UUID** — the ID provided to you earlier by the FeeCutEx representative.
