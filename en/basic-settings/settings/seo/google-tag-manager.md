# Google Tag Manager

Google Tag Manager (GTM) is a free tool from Google that allows you to manage tracking codes and place marketing tags on your website without needing to modify the website's actual code.

You can insert the code provided by Google using a hook, following the [instructions here](https://premium.gitbook.io/main/en/basic-settings/faq/kak-ustanovit-onlain-chat#ustanovka-s-pomoshyu-khuka-lyuboi-onlain-chat), or directly in your theme's file at `/wp-content/themes/your_theme_name/header.php` (within the or sections).
