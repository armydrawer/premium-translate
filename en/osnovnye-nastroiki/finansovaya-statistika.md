# Financial Statistics

This section is designed to display statistics for completed requests.

## Version 2.6

Available types of statistics:

<figure><img src="../.gitbook/assets/image (1771)_eng.png" alt=""><figcaption></figcaption></figure>

#### • By Exchange Amount:

<figure><img src="../.gitbook/assets/image (1764)_eng.png" alt="" width="477"><figcaption><p>Section Settings</p></figcaption></figure>

<figure><img src="../.gitbook/assets/image (409)_eng.png" alt="" width="260"><figcaption><p>Example Result</p></figcaption></figure>

**Start Date** — the initial date for the filter

**End Date** — the final date for the filter

**Currency** — select the currency for displaying statistics

**Currency Code** — select the currency code for displaying statistics

**Convert to** — convert all amounts to the selected currency

**Custom Central Bank Rate** — manually specified central bank rate

**Multiply by Custom Rate** — multiply data by the rate in the "**Custom Central Bank Rate**" field

**Include Partner Payments** — account for partner payments from filtered requests when calculating statistics

**Include Reserve Adjustments** — account for reserve adjustments in the selected currency when calculating statistics

**Exchange Direction** — select the exchange direction for calculating the total amount
