# Удаление Electrum с сервера

1.  В панели управления ISP Manager через раздел "**Сайты**" выделите нужный сайт кликом мыши и нажмите на пункт **"Войти как владелец"**.<br>

    <figure><img src="../../.gitbook/assets/изображение (71) (1).png" alt=""><figcaption></figcaption></figure>
2.  Откройте раздел **"Менеджер файлов"**.<br>

    <figure><img src="../../.gitbook/assets/изображение (67) (1).png" alt="" width="330"><figcaption></figcaption></figure>
3.  Удалите папку **Electrum**.<br>

    <figure><img src="../../.gitbook/assets/изображение (91).png" alt="" width="375"><figcaption></figcaption></figure>
4.  Откройте раздел **"Планировщик Cron".**<br>

    <figure><img src="../../.gitbook/assets/изображение (19) (1).png" alt="" width="305"><figcaption></figcaption></figure>
5. Удалите задания cron, созданные для Electrum.

<figure><img src="../../.gitbook/assets/image (1450).png" alt=""><figcaption></figcaption></figure>

6. <mark style="color:red;">Обязательно перезагрузите сервер</mark>.
