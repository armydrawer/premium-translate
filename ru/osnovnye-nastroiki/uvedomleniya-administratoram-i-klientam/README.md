# Уведомления администраторам и клиентам

В скрипте есть возможность отправлять уведомления пользователям и сотрудникам обменного пункта по 3 каналам:

* [Telegram](https://premium.gitbook.io/main/osnovnye-nastroiki/uvedomleniya-administratoram-i-klientam/uvedomleniya-v-telegram)
* [E-mail](https://premium.gitbook.io/main/osnovnye-nastroiki/uvedomleniya-administratoram-i-klientam/uvedomleniya-po-e-mail)
* [SMS](https://premium.gitbook.io/main/osnovnye-nastroiki/uvedomleniya-administratoram-i-klientam/uvedomleniya-po-sms)

Выберите подходящий вам вариант и настройте отправку уведомлений по инструкции.
